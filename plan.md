# HLB 보컬스튜디오 — 요금제 + 커리큘럼 재구성 + 기능 완성

## 배경
오프라인 보컬 레슨 구조를 온라인으로 옮긴다. 무료로 맛보기(이완~두성밸런스) → 진짜 변화를 원하면 발성전문반 결제(세팅~가창). 선생님 직접 피드백은 건당 유료.

## 요금제 구조 (확정)

| 플랜 | 가격 | 콘텐츠 범위 | 해금 방식 |
|------|------|------------|----------|
| 무료 | 0원 (광고) | 1~18단계 + 스케일자율 + 내장변환 | 자유 진행 (채점 없음) |
| 발성전문반 | 15만원/월 | 전체 28단계 + AI 커버 + 성장리포트 + 4축분석 | 채점 기반 해금 |
| 유료 피드백 | 5만원/회 | 선생님 직접 듣고 진단 | 건당 구매 |
| 정액제 | 19,900원/월 | 광고 제거 + 모든 컨텐츠 자유이용 (코칭 해금 제외) | 자유 이용 |

### 무료 콘텐츠 상세
- 진도: 1~18단계까지 자유 진행 (이완기초→호흡허밍→모음순화→발음확장→두성밸런스)
- 피아노 스케일 연습: 자율 이용, 일정 시간 후 광고
- 목소리 변환: 내장 프리셋만
- AI 코치 채팅: 기본 템플릿 피드백
- 채점/해금: 없음 (자유 진행)

### 발성전문반 콘텐츠 상세
- 무료 콘텐츠 전부 포함 + 광고 없음
- 19~28단계 채점 기반 해금 (세팅훈련→두성심화→고음표현→실가창)
- 4축 긴장 분석 (후두/혀뿌리/턱/성구전환) 상세 리포트
- 각 단계별 매커니즘 AI 해설 ("왜 이 연습을 하는지")
- 1:1 맞춤 커리큘럼 + 루틴 설계
- 일간 + 주간 성장 리포트
- AI 커버 무제한
- 곡 연습 + 구간 분석

### 정액제 콘텐츠 상세
- 광고 제거
- 모든 컨텐츠 자유 이용 (스케일, AI 커버, 곡 연습 등)
- 코칭 해금은 없음 (18단계까지만)
- 가격: 19,900원/월

### 유료 피드백
- 녹음 제출 → 선생님(HLB) 직접 청취
- AI 진단 결과 + 선생님 소견 비교
- 구체적 개선 방향 제시
- 무료/발성전문반 모두 추가 구매 가능

## 커리큘럼 구조 (28단계)

### 무료 영역 (1~18단계) — "맛보기: 목소리가 달라지는 체험"

**Block 1: 이완 기초 (1~3)**
1. 설근 안정화 — 혀/목/턱 이완, '어' 라운딩
2. 후두 안정 — 혀 내려가면 후두 안정, '어' 라운딩
3. 이완 라운딩 — 이완 유지하며 스케일 연결

**Block 2: 호흡/허밍 (4~6)**
4. 복식호흡 기초 — 배 압력 감각 인식 (피아노 없음)
5. 허밍 세팅 — 성대 접촉 기본 감각, '흠' 라운딩
6. 백프레셔 허밍 — 허밍 + 복부 압력 연결

**Block 3: 모음 순화 (7~9)**
7. 모음 5개 라운딩 — 턱 안정 유지, '이에아오우'
8. 모음 연결 — 음색 균일성, '에이오우'
9. 모음 조합 — 5모음 하나의 흐름으로

**Block 4: 발음 확장 (10~13)**
10. 옴 허밍 — 혀 안정 유지, '옴'
11. 멈 체킹 — 성대 접촉 감각, '멈'
12. 국 두성밸런스 — 두성 영역 진입, '국'
13. 1.5옥타브 스케일 — 넓은 음역 연결

**Block 5: 두성/연결 앞부분 (14~18)**
14. 복압 바람불기 — 복부 압력 활성화 (피아노 없음)
15. 허 복부 반응 — 소리와 복부 연결, '허'
16. 성대 접촉 세팅 — 가벼운 성대 접촉, '음'
17. 세팅 유지 스케일 — 1.5옥타브 세팅 유지
18. 두성 밸런스 — 최소 힘으로 두성 소리, 트라이어드

→ **여기까지 하면**: "목에 힘 빼는 감각 + 복부 압력 + 두성 감각"을 체험
→ **마케팅 포인트**: "여기까지만 해도 목소리 달라진 게 느껴지죠? 세팅부터는 전문반에서."

### 발성전문반 영역 (19~28단계) — "세팅 → 고음 → 가창"

**Block 6: 두성 심화 + 연결 (19~22)**
19. 두성 선창 — 선생님 소리 따라하기
20. 성구전환 체험 — 이완으로 자연스러운 전환
21. 말자리 훈련 — 세팅 유지 + 발음 연결, '마미무메모'
22. 음정 연결 — 풀스케일 올라갔다 내려오기

**Block 7: 고음/표현 (23~26)**
23. 고음 세팅 — 옥타브 도약, 세팅 유지
24. 압력 강화 — 고음에서 복부 압력 유지
25. 빠사지오 연습 — 성구전환점 매끄럽게
26. 발음별 세팅 유지 — 가사 붙이기 준비, '나니누네노'

**Block 8: 실가창 (27~28)**
27. 말자리 실가창 — 실전 곡에 세팅 적용
28. 자유 표현 — 모든 훈련 종합

→ **결제 동기**: "세팅 없이 고음 가면 목 상합니다. 세팅부터 가창 적용까지 AI가 채점하며 해금."
→ **차별화**: 채점 기반 해금 = 오프라인 레슨에서 선생님이 "다음 단계 넘어가도 돼" 하는 것과 동일

## 마케팅 전략

### 무료 → 발성전문반 전환 포인트
- 18단계(두성밸런스) 완료 후 "축하합니다! 여기까지 목소리가 많이 달라졌죠?"
- "세팅 훈련부터는 AI가 채점하며 단계별로 해금합니다"
- "세팅 없이 고음으로 가면 목에 무리가 옵니다. 안전하게 성장하세요."
- 19단계 잠금 화면에서 발성전문반 가입 유도

### 유료 피드백 전환 포인트
- AI 진단 결과 화면에서 "더 정확한 진단을 원하시면 선생님에게 직접 받아보세요"
- 특정 단계에서 3회 이상 불통과 시 "선생님 피드백을 받아보시겠어요?"
